# LEIM_FSO_22_23
Repo da UC de FSO do curso de LEIM 2022/2023

Este projeto tem como base o estudo de Fundamento de Sistemas Operativos.
Programação feita em Java, com base em processos, tarefas, semáforos e canais de comunicação com transmissão de mensagens, através do robot EV3 da LEGO Mindstorms.
